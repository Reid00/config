# Personal Config

个人编辑器与开发工具配置文件集合。

## 目录

### Zed

[Zed](https://zed.dev/) 编辑器配置，位于 `Zed/settings.json`。

- **字体**: `.ZedMono`，编辑器 16px，UI 15px
- **主题**: One Dark Pro (暗色) / One Light (亮色)
- **快捷键**: VSCode 风格
- **终端**: PowerShell
- **自动保存**: 1 秒延迟
- **AI 模型**: 通过 OpenAI 兼容接口接入 qwen3.6-plus

#### Python

- 语言服务器: basedpyright
- Lint & 格式化: ruff (行宽 80, 自动组织 import)
